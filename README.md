# Nexus Productivity Dashboard

Nexus is a beautiful, biomorphic desktop-style productivity dashboard built with pure HTML, CSS, and Vanilla JavaScript. It features an adaptive grid layout, a dynamic JavaScript router for state-driven view rendering, and localized task storage.

## Features

- **Component Modularity**: Structured UI with a persistent sidebar, main workspace, and contextual modals.
- **Biomorphic Theme**: Deep dark backgrounds (`#0e0e12`), a glassmorphic sidebar, and soft glow accents for active states.
- **State-Driven Routing**: Seamless navigation between views (Dashboard, Upcoming Events, To-Do List, Pending Work, Calendar) without page reloads.
- **Persistent Local Data**: Uses `localStorage` to save and filter tasks by status (Pending/Completed).
- **Adaptive Grid**: Fully responsive layout that ensures functionality on both desktop and mobile devices.

## Project Structure

- `index.html`: Semantic HTML layout utilizing Flexbox.
- `style.css`: Comprehensive styling with CSS variables and responsive design principles.
- `script.js`: Core logic for the JS Router and task management via localStorage.

## Setup Instructions
